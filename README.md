# Trivia
Trivia game site, implemented in Node.js

## timing: 
11/22 3:49pm - 6:08pm (2hr 21min)
11/23 4:20pm - 5:20pm (1hr), 6:18pm - 6:57pm (39 min)

## Instructions:
- download repository 
- run with `npm run dev` 
- enter `o` in the command line.

### Debugging:
- ensure everything is properly installed with
    - `npm install vite`
    - `npm install react`
    - `npm install react-router-dom`
    - `npm init y`
